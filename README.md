# real_estate_simulator

不動産投資の税金対策・キャッシュフロー・売却時手残りを確認する Streamlit アプリです。

## 主な機能

- 購入価格、売却価格、家賃、金利、ローン年数などの入力
- 運用中キャッシュフローの計算
- 減価償却の計算
- 運用中の税効果の計算
- 譲渡所得と譲渡所得税の計算
- 売却時手残りの計算
- 最終利益の計算
- 年次キャッシュフロー表とグラフ表示

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 設定

- Repository: `wenqian3985-gif/real_estate_simulator`
- Branch: `main`
- Main file path: `app.py`

## 注意

本アプリは概算シミュレーション用です。実際の税務判断は税理士等に確認してください。
